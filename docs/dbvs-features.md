# DBGODVS 功能需求清单（基于 SVN 过滤）

> 从 SVN 完整功能集中，去除 DBGODVS 不需要的功能，保留并扩展 DBGODVS 所需的功能。
> DBGODVS 定位：本地单机个人版本管理 + 局域网小团队协同，无需安装 SVN/Git。

---

## 过滤原则

### 去除的 SVN 功能类别
| 去除类别 | 原因 |
|----------|------|
| 企业级认证（LDAP/AD/SSL 证书） | 个人/小团队使用，不需要企业认证 |
| 高级服务器配置（svnserve、WebDAV） | 内置 HTTP 服务器，不需要独立 SVN 服务 |
| BDB 后端存储 | FSFS 式文件存储即可 |
| externals 外部仓库引用 | 个人项目极少使用 |
| changelist 分组 | 简化操作，减少学习成本 |
| 属性管理（svn:executable、svn:mime-type等） | 不需要 POSIX 权限管理 |
| sparse checkout 稀疏检出 | 个人项目全量检出即可 |
| 钩子脚本 | 个人工具不需要服务端钩子 |
| 备份/恢复（dump/load） | 可通过文件系统备份实现 |
| 仓库迁移/升级 | 单机工具，无迁移场景 |

### DBGODVS 新增的 SVN 没有的功能
| 新增功能 | 原因 |
|----------|------|
| 从 GitHub/GitLab 拉取项目 | 现代开发必需 |
| 可视化 GUI 操作面板 | 降低使用门槛 |
| Monaco 代码差异查看器 | 比命令行 diff 更直观 |
| AI 友好的 CLI | 适配 AI 辅助开发工作流 |
| 一键安装 | 无需配置即可使用 |
| Windows 右键菜单集成 | 资源管理器内直接操作 |
